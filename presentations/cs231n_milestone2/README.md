# CS231N Milestone 2 Presentation

This folder is self-contained for Overleaf import.

## Files

- `main.tex` - Beamer source for the Milestone 2 technical approach deck.
- `figures/` - Local PNG assets used by `main.tex`.

## Overleaf

1. Upload this folder as a new Overleaf project, or zip the folder and upload the zip.
2. Set `main.tex` as the main document.
3. Compile with `pdfLaTeX`.

The deck follows the same basic format as the Milestone 1 deck:

- 16:9 Beamer slides
- `metropolis` theme
- Stanford red accent color
- local image assets with relative paths only
- inline bibliography, so no `.bib` import step is required
